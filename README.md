# c64-joystick-tester
Test your joystick input on the commodore 64. Written in [C64 Turbo Macro Pro](http://turbo.style64.org)

to build
```
tmpx  -i jstick.asm -o jstick.prg
```

to run in vice
```
x64 jystick.prg
```

to run in emulator
```
SYS 828
```

run/stop + restore to break out of program.
